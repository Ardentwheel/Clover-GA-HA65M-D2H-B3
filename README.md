# Clover EFI For GIGABYTE GA-HA65M-D2H-B3 (rev. 1.0)


 - Intel Core i3-2105M 
 - GA-HA65M-D2H-B3 (H61)
 - Memory 4.0 GB 1333 MHz DDR3
 - Intel(R) HD Graphics 3000
 - Realtek ALC889
 - Realtek RTL8111E chip (10/100/1000 Mbit)

---

#### Functional
- [X] CPU Turbo
- [x] Audio
- [x] Wried Network
- [ ] Wireless Network
- [ ] Bluetooth
- [x] AppleUSBMouse
- [x] AppleUSBKeyboard


#### How To Use
Copy EFI folder to EFI desk.
